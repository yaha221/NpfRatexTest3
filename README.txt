Данный проект построен на следующих технологиях с описанием причин выбора:

Язык С# - Последнии курсы колледжа с изучением основ ООП проходили на этом языке, плюс диплом так же писался на этом языке.
Плюсом можно добавить что множество приятных мелочей (к примеру сборщик мусора и т.д.) есть на С#, в отличии от С++. С плюсами знаком, но не сильно, первые два курса колледжа с ним проходил.

WPF - Более актуальная платформа нежели WinForms. В колледже учили работать с WinForms, которые не понравились из-за неудобства построения UI.
Так же множество приятных функций имеется только в WPF.

MS SQL Server - занимает мало места и вмещает в себя весь неоходимый функционал для данного задания.

ADO.Net - Удобство в использовании и настройке для работы с различными БД.

Паттерн MVVM - Для удобной навигации и разработки проекта.

Visual Studio 2022 - Поддержка платформы .NET 6.0
